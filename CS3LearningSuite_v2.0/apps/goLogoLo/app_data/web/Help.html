Help

The @goLogoLo is a desktop Java application for creating and editing logos. Users can add different types of elements to the logo and manipulate them to create a beautiful logo.

Step 1: Place your background!

To start, you may want to add a rectangle element as your background. You can simply use the Add Rectangle button in the right function pane.

Step 2: Add other things(and also remove them)!

Try to click on the other buttons in the adding bar. As you can see, there are five different types of elements that you can add. You can also click the Delete Items button to delete the item or items that you have selected.

Step 3: Change Layer Name!

Now you will have some elements in the workspace and they are named by default as "Layer x". For better recognition, you may want to change the name of elements. (Yes, you can still know which layer the element is at in Order column.) Click the Rename button below the table and enter the new name in the pop out dialog.

Step 4: Change Items' Layer!

By clicking the Move Up and Move Down buttons will let you to change the layer of the element within the logo.

Step 5: Save the LOGO!

Once you've completed your work, you can press the save button and give your file a name. Note that the name in the app text field will be the name used in the Recent Work list of the Welcome dialog.

Step 6: Export your To Do List

Press the Export button when you wish to export your logo to a Web page summarizing the list in full.
